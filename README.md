# マイ辞書 📖

読書中に出会った知らない単語を、その場でClaudeに質問して記録していく個人辞書です。
意味・使用例・おもしろ雑学、（植物や動物など実物があれば）写真つきで残ります。

## 使い方

### 1. 単語を質問する（基本は音声入力）
このフォルダで Claude Code を開いた状態で、知らない単語を自然文で質問します。

- 🎙️ 声で「**アケビってどういう意味？**」「**“逢魔が時”って何？**」
- 説明でもOK：「**山に生えてる紫色の実で、秋に裂けるやつ何だっけ**」
- タイプで素早く追加：`/dict アケビ @ 風の谷のナウシカ`（`@` 以降は出典）

Claude が意味・読み・使用例・雑学を調べ、実物なら写真を `images/` に保存し、`entries.js` に自動で追記します。漢字や同音異義語で紛らわしいときだけ「どの単語？」と確認します。

### 2. 辞書を見る
`index.html` をブラウザで開くだけ（ダブルクリックでOK／登録後に「ビューを開きますか？」で自動オープンも可）。

- 🔍 単語・読み・意味・タグを横断検索
- 🏷️ タグをクリックして絞り込み（複数選択可）
- 🗂️ 新しい順／古い順／読み順で並び替え
- カードをクリックすると、くわしい意味・使用例・雑学・写真・出典を表示

## ファイル構成

| ファイル | 役割 |
|---|---|
| `index.html` | 辞書ビューア（ブラウザで開く） |
| `styles.css` / `app.js` | ビューアの見た目とロジック（Alpine.js + PicoCSS） |
| `entries.js` | 辞書データ本体（`/dict` で自動追記。手編集も可） |
| `images/` | ダウンロードした実物写真 |
| `CLAUDE.md` | 質問→登録の振る舞いを定義（このフォルダで作業中に有効） |

> ビューアは CDN（PicoCSS・Alpine.js・Google Fonts）を読み込むため、初回表示にはネット接続が必要です。

## 写真のライセンスについて
写真は主に Wikipedia / Wikimedia Commons から取得し、各エントリに帰属（クレジット）を記録しています。個人利用の範囲で参照してください。
